Example Angular project to list, add, and edit contacts.

This project uses Angular 15 with bootstrap frontend toolkit

The following design patterns were used in the creation of this example
  • Feature Modules
  • Components
  • Template-driven Form
  • Services
  • Routing
  • Data Model
  • Dependency Injection
  • Lazy-loading feature modules
  • Use Router Resolver to pre-fetch component data prior to routing
  • RxJs Observables
